## 欢迎来到我的博客

你好！欢迎来到我的个人网站。这里是我记录学习心得、分享成长感悟的地方。

### 关于这个网站

这个网站使用 **Next.js** 构建，采用静态导出部署在腾讯云上。主要技术栈包括：

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- MDX（支持 Markdown 和 React 组件）

### 你可以在这里找到

1. **博客文章** - 我的技术学习笔记和思考
2. **项目分享** - 我参与或开发的有趣项目
3. **关于我** - 了解更多关于我的信息
